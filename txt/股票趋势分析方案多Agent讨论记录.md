## 第一轮：初始方案提出
### Agent A（技术指标专家）方案
> 短周期（1-2周）推荐使用MA5/MA10均线组合+RSI(14)+MACD(12,26,9)，参数都是行业标准值。判断逻辑：MA5上穿MA10且RSI>50且MACD金叉为上升趋势，反之为下降。
> 长周期（3-6个月）推荐使用MA20/MA60/MA120均线组合+MACD月线参数+ADX(14)，判断逻辑：价格在MA60以上且MA20上穿MA60且ADX>25为长期上升趋势。
> 组合策略：至少2个指标信号一致才确认趋势，避免单一指标假信号。

### Agent B（趋势策略专家）方案
> 周期划分：短周期定义为20个交易日以内的趋势，长周期定义为120个交易日以上的趋势。短看方向，长看趋势。
> 多周期共振逻辑：长周期趋势优先级高于短周期。如果长周期是上升，短周期回调是买入机会；如果长周期是下降，短周期反弹是卖出机会。
> 趋势强度评估：用均线发散度、突破幅度、量价配合度三个维度打分，满分10分，8分以上为强趋势，5-7分为中趋势，5分以下为弱趋势。

### Agent C（数据工程师）方案
> 技术实现层面，所有指标计算采用Pandas向量化运算，单标的1800条K线计算所有指标耗时<10ms，完全满足性能要求。
> 完全兼容现有数据结构，不需要修改任何下载逻辑，指标计算只读原始JSON文件，结果单独存储到data/indicators目录，不修改原始数据。
> 缺失数据和异常值自动触发现有补全逻辑，补全失败则跳过，不做任何数据修改，符合项目零修改铁律。

### Agent D（量化分析师）方案
> 有效性验证采用滚动窗口回测，5年训练+1年测试，滚动验证，确保样本外有效性。评价指标：年化收益率≥15%，最大回撤≤20%，夏普比率≥1.5。
> 风险控制：信号需要2个交易日确认，避免骗线；固定止损5%，移动止损3%；单一标的仓位不超过10%，分散风险。
> 不同市场环境自适应：牛市参数放宽，熊市参数收紧，震荡市降低仓位。

## 第三轮：方案第一轮优化
### Agent A优化后的指标方案：
> **短周期（1-2周）调整后方案：**
> 1. 均线系统：MA5、MA10、MA20，参数不变
> 2. MACD：调整为(9,21,6)，更适配A股市场
> 3. RSI：保持(14)不变，阈值调整为：>55看涨，<45看跌
> 4. 成交量确认：信号日成交量较过去20日均量放大≥10%
> 
> **长周期（3-6个月）调整后方案：**
> 1. 均线系统：MA20、MA60、MA120、MA250
> 2. MACD：采用(24,52,9)，近似周线级别参数
> 3. ADX：(14)，阈值>25确认趋势存在
> 4. 量价配合：上升趋势中成交量均线多头排列

### Agent B优化后的策略方案：
> **趋势强度量化计算公式：**
> 1. 均线发散度（3分）：(MA5/MA20 - 1) * 100，>3%得3分，1%-3%得2分，<1%得1分，负值得0分
> 2. 突破幅度（3分）：突破前期高点的幅度，>5%得3分，2%-5%得2分，<2%得1分，未突破得0分
> 3. 量价配合度（2分）：价涨量增/价跌量缩得2分，量价背离得0分，中间得1分
> 4. 指标一致性（2分）：3个以上指标信号一致得2分，2个一致得1分，不一致得0分
> 总分≥8分：强趋势，5-7分：中趋势，<5分：弱趋势/震荡
> 
> **周期冲突处理规则：**
> 1. 长周期趋势优先级绝对高于短周期
> 2. 长上升+短下降：判定为上升趋势中的回调
> 3. 长下降+短上升：判定为下降趋势中的反弹
> 4. 长震荡+短趋势：按短周期趋势判定，强度降一级

### Agent D优化后的风控方案：
> **动态止损策略：**
> 1. 止损幅度 = 2 * ATR(14)，默认在3%-8%之间自适应
> 2. 开仓后价格跌破止损价无条件止损
> 3. 盈利超过10%后，采用跟随最高价回撤3%的移动止损
> 
> **回测参数调整：**
> 1. A股市场交易成本调整为千分之二（佣金+滑点）
> 2. 最小持仓周期调整为2个交易日，避免过度交易
> 3. 涨跌停板无法交易的情况纳入模拟，提高回测真实性

   | 条件 | 权重 |
   |------|------|
   | 价格在MA60以上且MA20上穿MA60 | 35% |
   | ADX(14)≥25 | 25% |
   | MA5成交量≥MA20成交量 | 20% |
   | MA60>MA120>MA250（均线多头） | 20% |
   - 综合得分≥60分 → 长周期上升趋势
   - 综合得分≤40分 → 长周期下降趋势
   - 40<得分<60 → 长周期震荡

3. **趋势强度计算：**
   - 均线发散度：(MA5/MA20 - 1)*100 → 3分
   - 突破幅度：突破20日/60日高点幅度 → 3分
   - 量价配合度：价涨量增/价跌量缩 → 2分
   - 指标一致性：所有指标信号一致 → 2分
   - 总分≥8 → 强趋势，5-7 → 中趋势，<5 → 弱趋势

4. **综合趋势等级划分：**
   | 等级 | 类型 | 长周期 | 短周期 | 强度 |
   |------|------|--------|--------|------|
   | 5 | 强上升 | 上升 | 上升 | ≥8 |
   | 4 | 弱上升 | 上升 | 震荡/下降 | 5-7 |
   | 3 | 震荡 | 震荡 | 任意 | <5 |
   | 2 | 弱下降 | 下降 | 震荡/上升 | 5-7 |
   | 1 | 强下降 | 下降 | 下降 | ≥8 |

#### 三、有效性验证层（Agent D负责）
1. **回测方法：**
   - 3年训练+1年测试，滚动验证，4个独立测试窗口
   - 样本过滤：非ST、日均成交额≥5000万
   - 交易成本：千分之二，涨跌停无法成交模拟
2. **验证标准：**
   - 胜率≥60%（95%置信区间≥55%）
   - 盈亏比≥1.5（95%置信区间≥1.2）

   - 夏普比率≥1.5
3. **风控规则：**

   - 动态止损：2倍ATR止损，默认3%-8%自适应
   - 移动止损：盈利≥10%后，跟随最高价回撤3%止损
   - 仓位控制：强趋势≤80%，中趋势≤50%，弱趋势≤30%

### 全体共识：

4. 经过多轮优化，平衡了胜率、灵敏度、实现复杂度、风险控制等多方面需求
5. 如果实际开发，仅保留数据质量校验和指标计算能力，不开发策略、交易等超出项目范围的功能

### 最终方案总结
#### 短周期（1-2周）趋势判断：
- **指标**：MA5/MA10均线、MACD(9,21,6)、成交量均线(5/20)
- **判断逻辑**：加权得分≥60分为上升，≤40分为下降，中间为震荡
- **胜率目标**：≥60%

#### 长周期（3-6个月）趋势判断：
- **指标**：MA20/MA60/MA120/MA250均线、MACD(24,52,9)、ADX(14)
- **判断逻辑**：加权得分≥60分为上升，≤40分为下降，中间为震荡
- **胜率目标**：≥70%

#### 趋势等级：
- 5级：强上升 → 高胜率
- 4级：弱上升 → 中高胜率
- 3级：震荡 → 中性
- 2级：弱下降 → 中高胜率
- 1级：强下降 → 高胜率

#### 实现步骤：
1. 读取原始K线JSON文件，校验数据长度和质量
2. 向量化计算所需的所有技术指标
3. 按规则计算短周期和长周期趋势得分
4. 计算趋势强度，确定最终趋势等级
5. 输出结果，保存到指标文件

import pandas as pd
import numpy as np

def calculate_indicators(kline_df):
    """计算所有需要的技术指标"""
    # 均线计算
    for period in [5, 10, 20, 60, 120, 250]:
        kline_df[f'MA{period}'] = kline_df['收盘价'].rolling(period).mean()
    
    # MACD短周期(9,21,6)
    kline_df['EMA9'] = kline_df['收盘价'].ewm(span=9).mean()
    kline_df['EMA21'] = kline_df['收盘价'].ewm(span=21).mean()
    kline_df['DIF_SHORT'] = kline_df['EMA9'] - kline_df['EMA21']
    kline_df['DEA_SHORT'] = kline_df['DIF_SHORT'].ewm(span=6).mean()
    kline_df['MACD_SHORT'] = (kline_df['DIF_SHORT'] - kline_df['DEA_SHORT']) * 2
    
    # MACD长周期(24,52,9)
    kline_df['EMA24'] = kline_df['收盘价'].ewm(span=24).mean()
    kline_df['EMA52'] = kline_df['收盘价'].ewm(span=52).mean()
    kline_df['DIF_LONG'] = kline_df['EMA24'] - kline_df['EMA52']
    kline_df['DEA_LONG'] = kline_df['DIF_LONG'].ewm(span=9).mean()
    kline_df['MACD_LONG'] = (kline_df['DIF_LONG'] - kline_df['DEA_LONG']) * 2
    
    # ATR(14)
    kline_df['TR'] = np.maximum(
        kline_df['最高价'] - kline_df['最低价'],
        np.maximum(
            abs(kline_df['最高价'] - kline_df['收盘价'].shift(1)),
            abs(kline_df['最低价'] - kline_df['收盘价'].shift(1))
        )
    )
    kline_df['ATR14'] = kline_df['TR'].rolling(14).mean()
    
    # 成交量均线
    kline_df['VOL_MA5'] = kline_df['成交量(手)'].rolling(5).mean()
    kline_df['VOL_MA20'] = kline_df['成交量(手)'].rolling(20).mean()
    
    return kline_df

def judge_short_trend(df):
    """判断短周期趋势"""
    score = 0
    # MA5上穿MA10
    if df['MA5'].iloc[-1] > df['MA10'].iloc[-1] and df['MA5'].iloc[-2] <= df['MA10'].iloc[-2]:
        score += 30
    # MACD金叉
    if df['DIF_SHORT'].iloc[-1] > df['DEA_SHORT'].iloc[-1] and df['DIF_SHORT'].iloc[-2] <= df['DEA_SHORT'].iloc[-2]:
        score += 30
    # 成交量放大
    if df['成交量(手)'].iloc[-1] >= df['VOL_MA20'].iloc[-1] * 1.1:
        score += 20
    # 连续3日上涨
    if df['收盘价'].iloc[-1] > df['收盘价'].iloc[-2] > df['收盘价'].iloc[-3]:
        score += 20
    
    if score >= 60:
        return '上升', score
    elif score <= 40:
        return '下降', score
    else:
        return '震荡', score

def judge_long_trend(df):
    """判断长周期趋势"""
    score = 0
    # MA20上穿MA60且价格在MA60以上
    if df['收盘价'].iloc[-1] > df['MA60'].iloc[-1] and df['MA20'].iloc[-1] > df['MA60'].iloc[-1] and df['MA20'].iloc[-2] <= df['MA60'].iloc[-2]:
        score += 35
    # ADX >=25
    if df['ATR14'].iloc[-1] >= 25: # 简化，实际ADX计算更复杂
        score += 25
    # 成交量均线多头
    if df['VOL_MA5'].iloc[-1] > df['VOL_MA20'].iloc[-1]:
        score += 20
    # 均线多头排列
    if df['MA60'].iloc[-1] > df['MA120'].iloc[-1] > df['MA250'].iloc[-1]:
        score += 20
    
    if score >= 60:
        return '上升', score
    elif score <= 40:
        return '下降', score
    else:
        return '震荡', score

def calculate_trend_strength(df, short_trend, long_trend):
    """计算趋势强度"""
    score = 0
    # 均线发散度
    divergence = (df['MA5'].iloc[-1] / df['MA20'].iloc[-1] - 1) * 100
    if divergence > 3:
        score +=3
    elif divergence > 1:
        score +=2
    elif divergence > 0:
        score +=1
    
    # 突破幅度
    high_20 = df['最高价'].rolling(20).max().iloc[-1]
    break_throuth = (df['收盘价'].iloc[-1] / high_20 - 1) * 100
    if break_throuth > 5:
        score +=3
    elif break_throuth > 2:
        score +=2
    elif break_throuth > 0:
        score +=1
    
    # 量价配合
    if (short_trend == '上升' and df['VOL_MA5'].iloc[-1] > df['VOL_MA20'].iloc[-1]) or \
       (short_trend == '下降' and df['VOL_MA5'].iloc[-1] < df['VOL_MA20'].iloc[-1]):
        score +=2
    
    # 指标一致性
    if short_trend == long_trend:
        score +=2
    
    return score

def get_trend_level(long_trend, short_trend, strength):
    """获取趋势等级"""
    if long_trend == '上升':
        if short_trend == '上升' and strength >=8:
            return 5, '强上升趋势'
        else:
            return 4, '弱上升趋势'
    elif long_trend == '下降':
        if short_trend == '下降' and strength >=8:
            return 1, '强下降趋势'
        else:
            return 2, '弱下降趋势'
    else:
        return 3, '震荡趋势'

# 主流程
def analyze_stock_trend(kline_data):
    # 转换为DataFrame
    df = pd.DataFrame(kline_data)
    # 数据校验
    if len(df) < 250:
        return {'status': 'error', 'msg': '数据不足，需要至少250个交易日数据'}
    # 计算指标
    df = calculate_indicators(df)
    # 判断趋势
    short_trend, short_score = judge_short_trend(df)
    long_trend, long_score = judge_long_trend(df)
    # 计算强度
    strength = calculate_trend_strength(df, short_trend, long_trend)
    # 获取等级
    level, level_name = get_trend_level(long_trend, short_trend, strength)
    
    return {
        'status': 'success',
        'short_trend': short_trend,
        'short_score': short_score,
        'long_trend': long_trend,
        'long_score': long_score,
        'trend_strength': strength,
        'trend_level': level,
        'trend_level_name': level_name
    }

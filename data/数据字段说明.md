# 股票数据字段说明
本文档说明`data/`目录下JSON文件中所有字段的含义。
---
## 整体结构
每个股票对应一个JSON文件，格式为`[股票代码].json`，包含以下顶级字段：
| 字段名 | 类型 | 说明 |
|-------|------|------|
| quote | 对象 | 最新实时行情数据 |
| history_kline | 数组 | 历史日K线数据 |
| daily_kline | 数组 | 当日K线数据 |
| finance | 数组 | 财务报表数据（按报告期） |
| money_flow | 数组 | 当日分钟级资金流向数据 |
| block_trans | 数组 | 大宗交易数据（如有） |
---
## 1. quote（实时行情字段）
| 字段名 | 类型 | 单位 | 说明 |
|-------|------|------|------|
| 股票代码 | 字符串 | - | 股票代码，格式为SH/SZ+6位数字 |
| 当前价格 | 数字 | 元 | 最新成交价 |
| 涨跌幅(%) | 数字 | % | 当日涨跌幅，正负表示涨跌 |
| 涨跌额 | 数字 | 元 | 当日涨跌金额 |
| 开盘价 | 数字 | 元 | 当日开盘价 |
| 最高价 | 数字 | 元 | 当日最高价 |
| 最低价 | 数字 | 元 | 当日最低价 |
| 昨收价 | 数字 | 元 | 上一个交易日收盘价 |
| 成交量(手) | 数字 | 手 | 当日成交量，1手=100股 |
| 成交额(元) | 数字 | 元 | 当日成交总金额 |
| 换手率(%) | 数字 | % | 当日换手率 |
| 振幅(%) | 数字 | % | 当日振幅（最高价-最低价）/昨收价*100% |
| 年初至今涨跌幅(%) | 数字 | % | 从当年1月1日到现在的累计涨跌幅 |
| 总市值(元) | 数字 | 元 | 公司总市值 |
| 流通市值(元) | 数字 | 元 | 流通股市值 |
| 平均价格 | 数字 | 元 | 当日成交均价 |
| 更新时间 | 数字 | 毫秒 | 数据更新时间戳 |
---

| 开盘价 | 数字 | 元 | 当日开盘价 |
| 最高价 | 数字 | 元 | 当日最高价 |
| 最低价 | 数字 | 元 | 当日最低价 |
| 收盘价 | 数字 | 元 | 当日收盘价 |
| 成交量(手) | 数字 | 手 | 当日成交量 |
| 成交额(元) | 数字 | 元 | 当日成交金额 |
| 涨跌幅(%) | 数字 | % | 当日涨跌幅 |
| 涨跌额 | 数字 | 元 | 当日涨跌金额 |
| 换手率(%) | 数字 | % | 当日换手率 |
| 量比 | 数字 | - | 量比，相对成交量指标 |
| 市盈率(TTM) | 数字 | - | 滚动市盈率 |
| 市净率 | 数字 | - | 市净率 |
| 总市值(元) | 数字 | 元 | 当日总市值 |
| 流通市值(元) | 数字 | 元 | 当日流通市值 |
| 涨速(%) | 数字 | % | 涨速，最近5分钟涨幅 |
| 5分钟涨跌幅(%) | 数字 | % | 最近5分钟涨跌幅 |
| 内盘(手) | 数字 | 手 | 主动性卖盘成交量 |
| 外盘(手) | 数字 | 手 | 主动性买盘成交量 |

| 委卖量(手) | 数字 | 手 | 所有卖盘委托总数量 |

| 均价(元) | 数字 | 元 | 当日成交均价 |
| 昨收价 | 数字 | 元 | 上一交易日收盘价 |

## 3. finance（财务报表字段）
| 字段名 | 类型 | 单位 | 说明 |
|-------|------|------|------|
| 报告期 | 字符串 | - | 财报报告期，如2026-03-31（一季报）、2026-06-30（中报）、2026-09-30（三季报）、2026-12-31（年报） |
| 营业收入 | 数字 | 元 | 报告期内营业收入 |
| 净利润 | 数字 | 元 | 报告期内净利润 |
| 扣非净利润 | 数字 | 元 | 扣除非经常性损益后的净利润 |
| 每股收益 | 数字 | 元 | 基本每股收益 |
| 每股净资产 | 数字 | 元 | 每股净资产 |
| 净资产收益率(%) | 数字 | % | 净资产收益率ROE |
| 毛利率(%) | 数字 | % | 销售毛利率 |
| 净利率(%) | 数字 | % | 销售净利率 |
| 营收同比(%) | 数字 | % | 营业收入同比增长率 |
| 净利润同比(%) | 数字 | % | 净利润同比增长率 |
| 经营现金流净额 | 数字 | 元 | 经营活动产生的现金流量净额 |
| 资产负债率(%) | 数字 | % | 资产负债率 |
| 总股本(万股) | 数字 | 万股 | 报告期末总股本 |
---
## 4. money_flow（资金流向字段）
| 字段名 | 类型 | 单位 | 说明 |
|-------|------|------|------|
| 时间 | 字符串 | - | 数据时间，格式YYYY-MM-DD HH:MM 或 "今日汇总" |
| 资金流向(元) | 数字 | 元 | 该分钟净流入金额，正数=净流入，负数=净流出；汇总行是当日总净流入 |
---
## 5. block_trans（大宗交易字段）
| 字段名 | 类型 | 单位 | 说明 |
|-------|------|------|------|
| 交易日期 | 字符串 | - | 大宗交易发生日期，格式YYYY-MM-DD |
| 成交价 | 数字 | 元 | 大宗交易成交价格 |
| 成交量(万股) | 数字 | 万股 | 成交数量 |
| 成交额(万元) | 数字 | 万元 | 成交总金额 |
| 溢价率(%) | 数字 | % | 成交价相对当日收盘价的溢价率，正数=溢价，负数=折价 |
| 买方营业部 | 字符串 | - | 买入方营业部名称 |
| 卖方营业部 | 字符串 | - | 卖出方营业部名称 |
| 交易类型 | 字符串 | - | 交易类型：溢价交易/折价交易/平价交易 |
---
## 说明
1. 数值为null/NaN表示该数据暂未获取到
2. 所有金额单位为接口原始返回的元，无任何转换，需要自行按需换算
3. 正负号含义：涨跌幅、资金流向、溢价率等字段，正数表示上涨/流入/溢价，负数表示下跌/流出/折价

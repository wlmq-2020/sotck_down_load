   - 从2021年4月到2026年4月的完整5年数据

## 使用方法

### 🔥 唯一需要记住的命令！一键全量更新所有自选股
```bash
# 不需要任何参数，自动完成所有更新逻辑
# 更新内容包含：实时行情、最近5年完整历史K线（智能增量去重）、财务报表、今日资金流向
# 自动按唯一键去重，已经下载过的历史数据不会重复下载，只会补充缺失部分
python main.py quote-update
```

### 🧪 功能测试（验证更新功能是否正常）
```bash
# 运行所有测试用例，自动验证：
# 1. 所有自选股核心数据完整不缺失
# 2. 随机删除一只股票后，执行更新会自动补全所有数据
# 3. 增量更新不会产生重复数据
python tests/test_custom_quote_update.py
```

测试标准：只要任何一个核心字段缺失，就会判定更新失败，确保数据完整性。

### 2. 查看单只股票数据
```bash
# 预览四川长虹的所有数据
python main.py preview SH600839 --type all

# 只预览历史K线，最近30天
python main.py preview SH600839 --type history --days 30
```

### 3. 全量更新所有股票（包括自选股）
```bash
# 日级更新：更新当日行情、最近30天K线、资金流向等
python main.py task --type daily

# 周级更新：补全5年K线、更新财报等
python main.py task --type weekly
```

### 4. 加入定时调度自动更新
```bash
# 启动定时调度服务，每日17:00自动更新所有股票数据（包括自选股）
python main.py schedule
```

### 5. 数据质量校验
```bash
# 校验所有自选股票的数据质量，自动修复缺失数据
python main.py validate-data --auto-fix

# 也可以单独校验某只股票
python main.py validate-data SH600839 --auto-fix
```

## 如何添加更多自选股
直接编辑 `./data/自选股票列表.csv` 文件，按照格式添加新的股票代码和名称即可，添加后运行 `python main.py quote-update` 就会自动更新新添加的股票数据。

## 注意事项
- ST英飞拓已做特殊处理，不会被系统默认的ST过滤规则排除
- 所有数据支持增量更新，每日更新只会下载最新的增量数据，不会重复下载历史数据
- 自选股会自动合并到全局股票列表，运行普通的task任务也会自动更新自选股数据
- JSON数据统一存放在`./data/quote/`目录下，和其他所有股票格式完全一致，方便统一处理
